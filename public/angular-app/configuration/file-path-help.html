File Paths
  • Local: \\DFS_NAME\OFFICE\PROJECTS\YEAR\PROJECT_NUMBER+NAME\FOLDER\SUBFOLDER\MODEL_NAME_CENTRAL.RVT
    Local: C:\OFFICE\PROJECTS\YEAR\PROJECT_NUMBER+NAME\FOLDER\SUBFOLDER\MODEL_NAME_CENTRAL.RVT


    Notes:
    - some companies prefer to use DFS(Distributed File System) path over a specific letter drive designation. Ex. \\GROUP\ instead of C:\.
    - whichever method you decide to use, please make sure that it is compliant with the Local File Path Regex settings. Each file path will have to pass/match at least one Regex pattern specified in Settings in order to be posted to MC.
    - You can define multiple different Regex patterns in settings that will be used to verify a file path. Only one has to pass for the file to be designated as "valid"
    - file path must have .RVT file extension

  • Revit Server: RSN://SERVER_NAME/PROJECT_NAME/DISCIPLINE/MODEL_NAME_CENTRAL.RVT


    Notes:
    - file path must start with RSN://
    - file path must have .RVT file extension

  • Cloud Model: BIM 360://FOLDER/MODEL_NAME.RVT


    Notes:
    - file path must contain :// and not start with RSN://
    - file path must have .RVT file extension
